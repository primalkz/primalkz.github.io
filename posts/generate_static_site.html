Getting Started

A simple static site generator using pandoc and jq to create a blog from markdown files.

What It Does

  • Converts markdown to HTML with pandoc
  • Extracts/saves metadata with jq for timestamps
  • Runs everything with ./generate.sh
  • Style everything by editing css/style.css

Setup

  1. Clone the repo:
git clone https://github.com/primalkz/bashite.git
cd bashite
  1. Configure your site: Edit config.env with your information:
# Personal information
NAME="your-name"
DESCRIPTION="optional description"

# Social links
GITHUB="https://github.com/yourusername"
INSTAGRAM="https://instagram.com/yourusername"
YOUTUBE_EDITS="https://youtube.com/@yourchannel"
YOUTUBE_MAIN="https://youtube.com/@yourchannel"
TELEGRAM="https://t.me/yourusername"
EMAIL="youremail@gmail.com"
  1. Make your content:
mkdir -p md/posts
touch md/index.md
touch md/about.md
  1. Generate:
./generate.sh

Post Format

Posts go in md/posts/ with this header:

On top of the post markdown file add the following comment to define post title and its keywords: 
<!--
     title: "My First Blog Post" 
     keywords: ["blog", "tutorial", "web"]
-->


## Introduction

Welcome to my first blog post! This is where you write your amazing content.

## Main Content

Here you can put your main ideas, tutorials, or thoughts. Use **bold** for emphasis and *italic* for highlights.

### Code Example
console.log("Hello, World!");

### Lists

- Item 1
- Item 2

## Conclusion

Thanks for reading!

Content Structure

index.md Format

Your homepage content goes in md/index.md. You can use HTML for special effects like hover images:

# Your Name

<img src="./img/profile.jpg">

Your introduction text goes here. You can use **bold**, *italic*, and other markdown formatting.

about.md Format

Your about page uses standard markdown:

## About Me

Write about yourself here. You can use:

- **Bold text** for emphasis
- *Italic text* for highlights
- Lists for organized information
- Code blocks for technical content

### My Skills

- JavaScript, Python, HTML/CSS
- React, Node.js, Git
- And more...

### Contact

Feel free to reach out!

Files You Need

  • md/index.md - homepage content
  • md/about.md - about page content
  • md/posts/ - blog posts - put your .md files here with the header format shown above
  • archive.html - auto-generated archive page - shows all your posts automatically, you don’t need to touch this unless you want to modify the archive layout
  • social.html - social links sidebar - you can remove social icons you don’t want from this file

Styling

Change css/style.css to customize the design.
