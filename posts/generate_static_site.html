Last modified: 2025-12-26 19:49:24

Getting Started

A simple static site generator using pandoc and jq to create a blog from markdown files.

What It Does

  • Converts markdown to HTML with pandoc
  • Extracts/saves metadata with jq for timestamps
  • Runs everything with ./generate.sh
  • Style everything by editing css/style.css

Setup

  1. Clone the repo:
git clone https://github.com/primalkz/primalkz.github.io.git
cd primalkz.github.io
  1. Make your content:
mkdir -p md/posts
touch md/index.md
touch md/about.md
  1. Generate:
./generate.sh

Post Format

Posts go in md/posts/ with this header:


## Content here

Your post content...

Files You Need

  • md/index.md - homepage
  • md/about.md - about page
  • md/posts/ - blog posts

Styling

Change css/style.css to customize the design.
